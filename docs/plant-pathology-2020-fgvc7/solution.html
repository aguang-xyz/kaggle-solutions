Two Real Targets¶

Actually there are only two real targets: rust and scab.

  • healthy = !rust & !scab
  • multiple_disease = rust & scab

So here, we do a simple conversion to make two new columns: real_rust and real_scab. Then we can train two models to deal with them individually.

In [6]:
train['real_rust'] = train['rust'] | train['multiple_diseases']
train['real_scab'] = train['scab'] | train['multiple_diseases']

fig, axes = plt.subplots(1, 2)

train[['image_id', 'real_rust']].groupby('real_rust').count().plot.bar(ax = axes[0])

Image Processing¶

Now we load all train images and pre-process the images followed by these steps:

  • rgb to gray
  • gaussian filter to reduce noise
  • sobel

Have a look at the first nine results.

In [ ]:
from skimage.io import imread
from skimage.color import rgb2gray
from skimage.filters import gaussian, sobel

images = []

for idx, row in train.iterrows():
    
    image = imread('./dataset/images/' + row['image_id'] + '.jpg')
    
    # image = sobel(gaussian(rgb2gray(image), sigma = 5))
    image = rgb2gray(image)
    
    images.append(image)

fig, axes = plt.subplots(3, 3, figsize=(14,10))

for idx in range(0, 9):
    axes[(idx // 3), idx % 3].imshow(images[idx])
